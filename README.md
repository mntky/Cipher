# [CTF] Cipher tools
---

####caesar

install

```
git clone https://github.com/mnkty/Cipher/caesar

go install ~/caesar/
//or
~/caesar/binary/caesar
```

example

```
caesar -a "Pb qdph lv pqwnb."
//----Shift 1 word----
//Qc reqi mw qrxoc.
//~~~~~~~~~~~~~~~~~~~~
//----Shift 23 word----
//My name is mntky.
//~~~~~~~~~~~~~~~~~~~~
//----Shift pcog ku opvma.
```
